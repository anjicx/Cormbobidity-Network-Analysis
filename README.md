# Comorbidity Network Analysis

This project builds and analyzes a directed comorbidity network using logistic regression results.

## Data
The analysis uses:
* beta1 values from logistic regression
* p-values of statistical significance

Source dataset: https://github.com/aaronab/comorbidity_networks
